Percobaan 1 = SB-LSTM-RNN
	Perlakuan 1 (Model 1 (M1)) = Base model, titik panas (univariate)
	Perlakuan 2 (Model 2 (M2)) = Model titik panas + Iklim
	Perlakuan 3 (Model 3 (M3)) = Model titik panas + ENSO
	
Percobaan 2 = SB-GRU-RNN
	Perlakuan 1 (Model 4 (M4)) = Base model, titik panas (univariate)
	Perlakuan 2 (Model 5 (M5)) = Model titik panas + Iklim
	Perlakuan 3 (Model 6 (M6)) = Model titik panas + ENSO

Mencari model terbaik
	Percobaan 1 berdasarkan iklim = Eror M1 - Eror M2
	Percobaan 1 berdasarkan ENSO = Eror M1 - Eror M3
	Percobaan 2 berdasarkan iklim = Eror M4 - Eror M5
	Percobaan 2 berdasarkan ENSO = Eror M4 - Eror M6
		* parameter NN dibuat berdasarkan GridSearchCV
	
	Hasil ini disebut model terbaik
